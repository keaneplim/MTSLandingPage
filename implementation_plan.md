# Implementation Plan: Mitra Trans Perkasa Landing Page

## 1. Project Overview
**Client:** Mitra Trans Perkasa (MTS)
**Industry:** Logistics & Warehousing
**Core Message:** "WE EXIST BECAUSE OF CUSTOMER CONFIDENCE"
**Primary Goal:** Establish a professional, reliable online presence for a Medan-based logistics leader.

## 2. Technical Stack
- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4 (Modern Industrial Palette)
- **Animations:** Framer Motion (Subtle entry effects)
- **Icons:** Lucide React
- **Typography:** Inter (Primary), Montserrat (Headings)

## 3. Design System
- **Brand Colors:**
  - Deep Blue: `#003366` (Authority/Stability)
  - Action Blue: `#007BFF` (Innovation/Progress)
  - Background: White / Light Blue Tint (`#E6F0FF`)
- **Logo Concept:** Geometric Hexagonal Shield with abstract 'M' path lines.
- **Responsiveness:** Fluid typography and layout scaling from mobile to ultra-wide displays.

## 4. Site Architecture
1. **Hero:** High-impact value proposition with CTA.
2. **Philosophy:** Vision, Mission, and Safety Commitment cards.
3. **Services:** Core capabilities (Transportation, Distribution, Research).
4. **Sustainability:** Environmental and social responsibility section.
5. **Contact:** Integrated lead generation form and location data.

## 5. Roadmap
### Phase 1: Completed
- Project scaffolding and theme configuration.
- Responsive Navbar and Footer implementation.
- Core content sections with professional copy.
- Fluid responsiveness audit.

### Phase 2: Upcoming
- **Asset Replacement:** Swap Unsplash placeholders for real fleet/warehouse photos.
- **Technical Specs:** Implement a modal/tab system for fleet dimensions and warehousing capacities.
- **Bilingual Support:** Add Indonesian (Bahasa) translation toggle.
- **SEO Optimization:** Final metadata tuning and sitemap generation.
